If you want to run this you need the following libraries:
- tkinter
- Qiskit
- numpy
- random
- Pillow

This project was made for the Brown Pre-College Summer Course introduction to Quantum Computing 2025. The code was made by Laith Al-Wir and Charmaine.

The basic idea is to use a third party Qubit to generate a random number between 1-38 using a Hadamard gate; then using Quantum teleportation to securely share this data to ensure fairness.
